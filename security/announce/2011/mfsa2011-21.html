Mozilla Foundation Security Advisory 2011-21

Title: Memory corruption due to multipart/x-mixed-replace images
Impact: Critical
Announced: June 21, 2011
Reporter: Jordi Chancel
Products: Firefox, Thunderbird, SeaMonkey

Fixed in: Firefox 5
  Firefox 3.6.18
  Thunderbird 3.1.11
  SeaMonkey 2.2

Description

Security researcher Jordi Chancel reported a crash on multipart/x-mixed-replace images due to memory corruption.

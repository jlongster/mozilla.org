If our main support page does not list the Mozilla product you need help with, the Mozilla community has set up support venues for other Mozilla software.

Some tips...

  • Be as specific as possible.
  • Include information about your operating system.
  • Include which version of the product you are using.
  • Include any add-ons you have installed.

Web forum

MozillaZine is a community-run message board that provides user support. You can search for answers to questions other users have already asked, or post a new question of your own.

  • MozillaZine Other Applications and Distributions Forum

Newsgroup

Mozilla hosts public newsgroups for user support discussion on the news.mozilla.org server. Please read the Mozilla Forum Etiquette page before posting. You can search the archive on Google Groups.

  • mozilla.support.other

Community mailing list

Mozilla hosts public mailing lists for user support discussion, which are bi-directionally mirrored with the newsgroups. Please read the Mozilla Forum Etiquette page before posting. You can search the archive on Google Groups.

  • support-other Mailing list

Real Time Chat (IRC)

You can talk to other users in real time by joining the #mozillazine channel on mozilla.org's IRC server (irc.mozilla.org) with an IRC client, such as Chatzilla.

  • #mozillazine IRC Channel

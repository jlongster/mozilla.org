Test Status of Mozilla Accessibility on Unix


Introduction:
This is our first version of mozilla with accessibility function. It is really a big milestone for our accessibility task force!
It contains our 90%+ coding work, including cross-platform accessibility code (nsIAccessible extension), Mozilla ATK implementation (MAI) and GTK2 porting. We will do a full test this week.

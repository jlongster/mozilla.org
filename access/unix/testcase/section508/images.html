Test Case Description for  Bitmap Images

Purpose: Verify associate labels with controls, objects, icons, and images.


Initial  Conditions:
  1. Labels name an object.(eg,icon names,window titles,button labels,edit fields).
  2. Bitmap image changes meaning during a program's execution.
  3. Have Screen Reader program installed.
Steps/Description:
  1. Position the mouse over objects: including icon names,window titles,button labels,edit fields and any other feature of a program that is intended to allow the user to perform some action.
  2. Run the software with a screen reader.
 Expected Results:
  1. Verify the labels are associated programmatically with the object in order for the assisitive technology to make them available to the user.                                                          
  2. Verify that all text in client windows can be spoken.

Test Case Description for  Focus Indicator

Purpose: Verify a visual focus indicator that moves among interactive objects as the input focus changes can be programmatically exposed to assistive technology.

Initial  Conditions:
  1. Put the mouse (or other pointing device )aside and exercise each feature of mozilla using the keyboard.
  2. Have Screen Reader installed.
  3. Have Screen Magnifier installed.
Steps/Description:
  1. Using only the keyboard, navigate through the software, especially menus, dialogs, forms and property boxes.
  2. Run Magnifier as you navigate through the software.
  3. Run a screen reader as you navigate through the software.
 Expected Results:
  1. Verify the visual focus is clearly visible and easy to identify when you tab or arrow to the next object.
  2. Verify Magnifier tracks the focus indicator and displays it in the Magnifier window.
  3. Verify screen reader reads tracking the focus indicator.

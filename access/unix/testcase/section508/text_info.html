Test Case Description for  Textual information

Purpose: Verify text is provided through standard system function calls or through an API which supports interaction with assistive technology


Initial  Conditions:
  1. Have Screen Reader program installed.
Steps/Description:
  1. Position the mouse over objects.
  2. Run the mozilla with a screen reader.
 Expected Results:
  1. Verify the minimum information  about the text (text content, input caret location and attributes) is provided.                                     
  2. Verify that all text in client windows can be spoken.

Test Cases of Section 508 Standards for Electronic and Information Technology

Maintained by Jessie Li

This part of test cases are based on the Section 508 technical standards. There are two test areas. And some of these test cases are combined with Assistive Technology, such as Screen Reader, Screen Magnifier, etc. Go into for the details.

  • Software Applications and Operating Systems (1194.21)
      • This is applicable for both software applications and operating systems. They address program features that must be contained in software for the product to meet the standards.


  • Web-based intranet and Internet Information and Applications (1194.22)
      • These standards provide the requirements that must be followed when producing web pages. Assistive technologies, such as screen readers, can be used to test these web pages.

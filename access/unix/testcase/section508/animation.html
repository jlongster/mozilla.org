Test Case Description for  Animation

Purpose: Verify at least one option is provided to display animation in a non-animated presentation mode.


Initial  Conditions:
  1. Preference has this kind of option to display animation in a non-animated presentation mode.
  2. Have Screen Reader program installed.
Steps/Description:
  1. Use the preference to stop the animation.
  2. Run mozilla with a screen reader.
 Expected Results:
  1. Verify that all animation can be stopped and all information is still conveyed.                                   
  2. Verify that all text in client windows can be spoken.

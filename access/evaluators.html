Information for Governments and Other Organizations Evaluating Mozilla

  1. Mozilla's Section 508 Compliance

    Section 508 is a set of accessibility requirements that help US federal agencies decide what software applications they can use.

  2. Mozilla's W3C User Agent Guidelines Evaluation Report

    W3C's WAI is advancing another important set of accessibility standards called UAAG. These guidelines go much further than Section 508 does and Mozilla is further from doing a good job at satisfying them.

Live Chat

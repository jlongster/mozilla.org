Direct 3D DLL

The Microsoft D3D9X DLL is an example of an application of the binary components policy. To provide a good WebGL experience to Windows users with Intel graphics cards we planned on shipping ANGLE, an open source library, which translates WebGL/OpenGL to Direct3D 9. However, one of the pieces that ANGLE depends on is the D3D9X library provided by Microsoft. This library is not available in default Windows installs. Microsoft permits either the creation of a 2.5-3MB separate installer that all Windows users would need to execute when installing Firefox (which would increase download time and complexity and be a poor user experience), or bundling the required 800k DLL with the product (as we have chosen to do).

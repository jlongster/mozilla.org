Τι νέο υπάρχει στον Thunderbird 3

Ελπίζουμε να απολαύσετε τα νέα χαρακτηριστικά του Thunderbird. Έχουμε νέα εργαλεία αναζήτησης, μηνύματα σε καρτέλες, αρχειοθέτηση μηνυμάτων και εκατοντάδες πρόσθετα για να μπορέσετε να κάνετε τον Thunderbird 3 πραγματικά δικό σας.

Αναζήτηση με εργαλεία προχωρημένου φιλτραρίσματος

Πληκτρολογήστε τον όρο που αναζητάτε στη γενική εργαλειοθήκη αναζήτησης και θα σας ανοίξουμε τα αποτελέσματα σε νέα καρτέλα. Μπορείτε να εντοπίσετε εύκολα αυτό που ψάχνετε με τα εργαλεία φιλτραρίσματος και το χρονοδιάγραμμα του Thunderbird.

Μηνύματα σε καρτέλες

Αν σας αρέσει η περιήγηση σε καρτέλες του Firefox, θα λατρέψετε την ανάγνωση ηλεκτρονικού ταχυδρομείου σε καρτέλες. Κάντε διπλό κλικ ή πατήστε το πλήκτρο «enter» πάνω σε ένα μήνυμα για να το ανοίξετε σε νέα καρτέλα. Μπορείτε να κάνετε δεξί κλικ πάνω σε μηνύματα ή φακέλους για να τα ανοίξετε σε μια καρτέλα στο παρασκήνιο.

Αρχειοθέτηση μηνυμάτων

Επιλέγοντας το κουμπί αρχειοθέτησης ή πατώντας το πλήκτρο «Α» θα αρχειοθετείται το μήνυμα σας. Η αρχειοθέτηση μηνυμάτων τα μεταφέρει από τα εισερχόμενα σε άλλο φάκελο του συστήματος αρχειοθέτησης.

Περισσότερα νέα χαρακτηριστικά για να εξερευνήσετε:

  • Έξυπνοι φάκελοι
  • Διαχειριστής δραστηριότητας
  • Βιβλίο διευθύνσεων με ένα κλικ
  • Υπενθύμιση συνημμένων
  • Νέος διαχειριστής πρόσθετων και εκατοντάδες νέα πρόσθετα για να δοκιμάσετε

Για περισσότερες πληροφορίες, επισκεφθείτε την αρχική σελίδα του Thunderbird.

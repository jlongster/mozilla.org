Σας ευχαριστούμε που χρησιμοποιείτε τον Thunderbird!

Τι νέο υπάρχει στον Thunderbird;

Ελπίζουμε να ευχαριστηθείτε από όλα τα νέα χαρακτηριστικά που έχει ο Thunderbird. Ο Thunderbird είναι ολόδικος σας και με εκατοντάδες διαθέσιμα πρόσθετα που σας επιτρέπουν να τον προσαρμόσετε όπως ακριβώς θέλετε.

Γρηγορότερος, ασφαλέστερος και πιο σταθερός

  • Ο Thunderbird έχει καλύτερες αποκρίσεις και είναι πιο γρήγορος στην εκκίνηση και την εκτέλεση του
  • Ο Thunderbird βασίζεται πάνω στη νέα μηχανή Mozilla Gecko 7
  • Διαθέτει αναρίθμητες βελτιώσεις σε όλες τις πλατφόρμες, που προσφέρουν βελτιωμένη ασφάλεια και σταθερότητα

Πιο εύκολο ξεκίνημα

  • Διαθέτει ανανεωμένο βοηθό δημιουργίας λογαριασμών για ακόμα πιο εύκολη προσθήκη λογαριασμών αλληλογραφίας. Θα τον βρείτε στο μενού «Αρχείο», επιλέγοντας «Νέο» κι έπειτα «Λογαριασμός αλληλογραφίας...».
  • Διαθέτει νέα σελίδα πληροφοριών αντιμετώπισης προβλημάτων για να σας βοηθάει όταν αντιμετωπίζετε προβλήματα στην αποστολή ή τη λήψη μηνυμάτων. Θα την βρείτε στο μενού «Βοήθεια», επιλέγοντας την «Αντιμετώπιση προβλημάτων».

Νέος διαχειριστής πρόσθετων

  • Επιλέγοντας τα «Πρόσθετα» από το μενού «Εργαλεία», μπορείτε να ξεκινήσετε την εξερεύνηση σας σε όλα τα διαθέσιμα πρόσθετα για τον Thunderbird.
  • Προσθέστε το Lightning Calendar
  • Προσθέστε το Conversation View, μια διαφορετική πρόταση παρουσίασης των μηνυμάτων σας.

Περισσότερα νέα χαρακτηριστικά που περιμένουν να τα ανακαλύψετε

  • Thunderbird is based on the new Mozilla Gecko 7 engine
  • Several user interface fixes and improvements
  • Several fixes to attachment handling
  • Ability to print a summary of selected email messages
  • Platform improvements to Address Book
  • Fixed several security issues
  • Numerous platform fixes that improve speed, performance and stability
  • Δείτε τις σημειώσεις έκδοσης για περισσότερες λεπτομέρειες

Merci d'avoir choisi Thunderbird !

Les nouveautés de Thunderbird

Nous espérons que vous aimerez les nouvelles fonctionnalités de Thunderbird. Vous pouvez personnaliser Thunderbird comme vous le voulez, et avec des centaines de modules complémentaires à essayer, Thunderbird est tout à vous.

Une prise en main plus facile

  • L'assistant de création de compte a été revu pour faciliter la création de vos comptes. Il se situe dans le menu « Fichier > Nouveau > Comptes courrier… »
  • Une nouvelle page d'information de dépannage à été créée pour vous aider lorsque vous rencontrez des problèmes pour l'envoi ou la réception de courrier. Vous la trouverez dans le menu « Aide > Informations de dépannage ».

Un nouveau gestionnaire de modules complémentaires

  • Dans le menu « Outils », sélectionnez « Modules complémentaires » pour commencer à parcourir le catalogue de tous les modules complémentaires disponibles pour Thunderbird.
  • Ajouter Lightning Calendar
  • Ajouter Conversation View, une manière différente de voir vos messages

Encore plus de nouvelles fonctionnalités à explorer

  • Les onglets peuvent maintenant être réorganisés et déposés dans une fenêtre différente
  • La taille des pièces jointes est maintenant affichée
  • Les plugins peuvent à présent être chargés dans des flux RSS
  • Plusieurs correctifs des thèmes pour Windows Vista et Windows 7
  • Plus de 390 correctifs améliorent la vitesse, les performances et la stabilité
  • Consulter les notes de version pour plus de détails

O que há de novo no Thunderbird?

Esperamos que goste de todas as novas funcionalidades do Thunderbird. Pode personalizar o Thunderbird como quiser, e com centenas de extras para experimentar, o Thunderbird é todo seu.

Rápido, seguro e mais estável

  • O Thunderbird tem uma melhor resposta e é mais rápido a iniciar e a usar
  • O Thunderbird é baseado no novo motor Mozilla Gecko 9
  • Reparações em diversas plataformas para melhorar a segurança e a estabilidade do produto

É mais fácil começar

  • O assistente de criação de conta foi revisto, para melhorar a configuração do correio electrónico. Encontre-o no Menu Ficheiro, seleccione Novo, Conta de correio...
  • Nova página de resolução de problemas para o ajudar a encontrar o problema quando envia ou recebe correio. Encontre-o no Menu ajuda, Informação de suporte.

Novo gestor de extras

  • No Menu Ferramentas, seleccione extras para começar a explorar os extras disponíveis para o Thunderbird.
  • Adicione o Calendário Lightning
  • Adicione Ver conversação uma maneira diferente de ver o correio electrónico

Mais novas funcionalidades

  • Melhor suporte para Personas ao compor mensagens de correio usando o Livro de Endereços
  • Melhor funcionamento dos anexos com o teclado
  • Novo sistema para o desempenho e dados de usabilidade para nos ajudar a melhorar o Thunderbird
  • Para mais, veja o artigo Novo no Thunderbird 9

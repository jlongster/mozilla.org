O que há de novo no Thunderbird?

Esperamos que disfrute de todas as novas funcionalidades do Thunderbird. Pode personalizar o Thunderbird como quiser e com centenas de extras para experimentar, o Thunderbird é todo seu.

Mais fácil começar

  • Assistente de criação de conta revisto, para melhorar a configuração do correio. Encontre-o no menu Ficheiro, seleccione novo, conta de correio...
  • Nova página de informação sobre resolução de problemas para o ajudar quando encontrar problemas ao enviar ou receber correio. Encontre no menu de Ajuda, informação sobre resolução de problemas.

Novo gestor de extras

  • No menu Ferramentas, seleccione extras para começar a explorar todos os extras disponíveis para o Thunderbird.
  • Adicione o Lightning Calendar
  • Adicione o Conversation View uma maneira diferente de ver o seu correio

Mais funcionalidades para explorar

  • Os separadores podem agora ser reordenados e arrastados para diferentes janelas
  • Os tamanhos dos anexos são agora apresentados juntamente ao anexo
  • Os plugins podem ser carregados em fontes RSS
  • Existem várias correcções sobre o tema para o Windows Vista e Windows 7
  • Mais de 390 correcções na plataforma que melhoram a velocidade, o desempenho e a estabilidade

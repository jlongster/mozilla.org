¡Gracias por usar Thunderbird!

¿Qué hay de nuevo en Thunderbird?

Esperamos que disfrutes de todas las nuevas funcionalidades de Thunderbird. Puedes modificar Thunderbird como quieras, y con cientos de complementos para probar, Thunderbird es completamente tuyo .

Más rápido, seguro y estable

  • Thunderbird responde mejor y es más rápido para iniciar
  • Thunderbird está basado en el nuevo motor Mozilla Gecko 6
  • Numerosas mejoras en la plataforma que aumentan la seguridad y estabilidad del producto

Más fácil para empezar

  • El asistente de creación de cuentas fue revisado para mejorar la configuración de correo electrónico. Podés verlo desde el menú Archivo, seleccionando Nueva -> Cuenta de correo…
  • Nueva página de información de problemas para ayudarte cuando tenés problemas al enviar o recibir correo electrónico. Podés verlo en el menú Ayuda -> Información para resolver problemas.

Nuevo administrador de complementos

  • Dentro del menú Herramientas, seleccioná Complementos para explorar todos los complementos disponibles para Thunderbird.
  • Agregá el Calendario Lightning
  • Agregá Conversation View una forma diferente de ver tu correo

Más funcionalidades nuevas para descubrir

  • Muchas mejoras al tema para Windows 7
  • Permite usar la lista de saltos de Windows 7
  • Muchas mejoras al importar correo desde Microsoft Outlook
  • La comprobación de cliente predeterminado funciona en las nuevas distribuciones de Linux
  • Otras mejoras en la interfaz de usuario
  • Mirá las notas del lanzamiento para más información

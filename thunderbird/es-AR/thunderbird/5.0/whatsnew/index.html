¡Gracias por usar Thunderbird!

¿Qué hay de nuevo en Thunderbird?

Esperamos que disfrutes de todas las nuevas funcionalidades de Thunderbird. You can customize Thunderbird however you want, and with hundreds of add-ons to try, Thunderbird is all yours.

Más fácil para empezar

  • El asistente de creación de cuentas fue revisado para mejorar la configuración de correo electrónico. Podés verlo desde el menú Archivo, seleccionando Nueva -> Cuenta de correo...
  • Nueva página de información de problemas para ayudarte cuando tenés problemas al enviar o recibir correo electrónico. Podés verlo en el menú Ayuda -> Información para resolver problemas.

Nuevo administrador de complementos

  • Dentro del menú Herramientas, seleccioná Complementos para explorar todos los complementos disponibles para Thunderbird.
  • Agregá el Calendario Lightning
  • Agregá Conversation View una forma diferente de ver tu correo (en inglés)

Más funcionalidades nuevas para descubrir

  • Las pestañas pueden ser reordenadas y arrastradas a ventanas diferentes
  • El tamaño de los adjuntos se muestra al lado de los mismos
  • Los plugins pueden cargarse en canales de RSS
  • Se hicieron muchos arreglos a los temas para Windows Vista y Windows 7
  • Más de 390 arreglos en la pataforma que mejoran la velocidad, el rendimiento y la estabilidad
  • Mirá las notas de la versión para tener más información

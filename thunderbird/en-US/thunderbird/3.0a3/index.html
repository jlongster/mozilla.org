Shredder Alpha 3 Preview Release.

Released: October 14, 2008

  • New Features and Changes
  • Downloading

Shredder Alpha 3 is a preview release for the next major version of Thunderbird that is being built on top of the next generation of Mozilla's layout engine, Gecko 1.9. Shredder Alpha 3 is being made available for testing purposes only and is intended for developers and our testing community.

New Features and Changes

New Features and changes in this milestone that require feedback include:

  • Improved Message Reader View

    This is the first stage of a series of improvements to refine the message reading experience; the first stage brings email controls closer to area in which the user is operating.

  • Add/Edit Contacts from the Message Header

    We now provide at-a-glance indication if someone is already in your address book, if they are not, you add them with one click. You can also edit contact details inline.

  • Integrated Vista Search

    Windows Vista users can now choose to see Thunderbird email and news messages in Windows search results.

  • Automatic Message Download for IMAP

    Thunderbird will now download IMAP messages in the background to allow for faster message loading, and better off-line operation. See the release notes for details on enabling this feature.

  • Birthday Field in Address Book

    You can now enter the dates of your contacts birthdays into the address book.

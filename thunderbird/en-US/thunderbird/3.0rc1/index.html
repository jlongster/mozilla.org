Thunderbird 3 
Release Candidate 1

Released: November 24, 2009

  • New Features and Changes
  • Downloading

Thunderbird 3 Release Candidate 1 is now available for download. While this milestone is considered to be stable, it is intended for developers and members of our testing community to use for evaluation and feedback. Users of this latest release version of Thunderbird should not expect all of their add-ons to work properly with this milestone.

New features in Thunderbird 3 that require feedback include:

  • New Search Tools
  • Tabbed Email
  • Message Archiving
  • New Mail Account Setup Wizard
  • Improvements for Developers

Downloading

Please read the release notes before downloading for more information about this release including known issues.

Testers can download Thunderbird 3 Candidate 1 builds for Windows, Mac OS X, and Linux in 43 different languages. Community chat about Thunderbird 3 can be followed at the Thunderbird 3 wiki and on irc.mozilla.org in #thunderbird.

Note: Please do not link directly to the download site. Instead we strongly encourage you to link to this Thunderbird 3 Release Candidate 1 milestone announcement so that everyone will know what this milestone is, what they should expect, and who should be downloading to participate in testing at this stage of development.

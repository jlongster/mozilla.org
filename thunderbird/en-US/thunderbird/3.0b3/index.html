Thunderbird 3 Beta 3
Preview Release

Released: July 21, 2009

  • New Features and Changes
  • Downloading

Thunderbird 3 Beta 3 is now available for download. This milestone is focused on testing the core functionality of the new features and platform changes that will be included in Thunderbird 3.

New features and changes in this milestone that require feedback include:

  • New Tabbed Email Improvements
  • New Message Summary
  • Improvements For Developers
  • see the complete list of bugs fixed

Downloading

Please read the release notes before downloading for more information about this release including known issues.

Testers can download Thunderbird 3 Beta 3 builds for Windows, Mac OS X, and Linux in 43 different languages. Community chat about Thunderbird 3 can be followed at the Thunderbird 3 wiki and on irc.mozilla.org in #thunderbird.

Note: Please do not link directly to the download site. Instead we strongly encourage you to link to this Thunderbird 3 Beta 3 milestone announcement so that everyone will know what this milestone is, what they should expect, and who should be downloading to participate in testing at this stage of development.

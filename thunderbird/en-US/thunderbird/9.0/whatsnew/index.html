What's New in Thunderbird?

We hope you enjoy all the new features in Thunderbird. You can customize Thunderbird however you want, and with hundreds of add-ons to try, Thunderbird is all yours.

Faster, Safer, and More Stable

  • Thunderbird is more responsive and faster to start up and use
  • Thunderbird is based on the new Mozilla Gecko 9 engine
  • Numerous platform fixes to improve security and product stability

Easier to Get Started

  • Revised account creation wizard, to improve email setup. Find it under the File Menu, select New, Mail Account...
  • New Troubleshooting Information page to help you when encountering trouble sending or receiving email. Find it under the Help Menu, Troubleshooting Information.

Great Add-ons

  • Under the Tools Menu, select Add-ons to start exploring all the available Add-ons for Thunderbird.
  • Add the Lightning Calendar
  • Add Conversation View a different way to view your email

More New Features to Explore

  • Better support for Personas when composing emails and using Address book
  • Better keyboard handling for attachments
  • New opt-in system for performance and usability data to help us make Thunderbird better
  • See New in Thunderbird 9 article for more

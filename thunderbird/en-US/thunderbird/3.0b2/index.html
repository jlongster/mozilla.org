Thunderbird 3 Beta 2
Preview Release

Released: February 26, 2009

  • New Features and Changes
  • Downloading

Thunderbird 3 Beta 2 is now available for download. This milestone is focused on testing the core functionality of the new features and platform changes that will be included in Thunderbird 3. For additional information about the new features, visit this blog post.

New features and changes in this milestone that require feedback include:

  • New Activity Manager
  • New ability to Archive email
  • see the complete list of bugs fixed

Downloading

Please read the release notes before downloading for more information about this release including known issues.

Testers can download Thunderbird 3 Beta 2 builds for Windows, Mac OS X, and Linux in 43 different languages. Ongoing planning for Thunderbird 3 can be followed at the Thunderbird 3 wiki and on irc.mozilla.org in #maildev.

Note: Please do not link directly to the download site. Instead we strongly encourage you to link to this Thunderbird 3 Beta 2 milestone announcement so that everyone will know what this milestone is, what they should expect, and who should be downloading to participate in testing at this stage of development.

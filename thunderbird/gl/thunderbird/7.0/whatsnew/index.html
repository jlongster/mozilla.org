Grazas por usar Thunderbird!

Que novidades ten o novo Thunderbird?

Esperamos que goce das novas características de Thunderbird. Vostede pode personalizar o Thunderbird canto queira, ademais pode probar centos de complementos. Thunderbird é todo seu.

Máis rápido, máis seguro e máis estábel

  • O Thunderbird é máis áxil e máis rápido, tanto no inicio como no uso normal
  • O Thunderbird está baseado no novo motor Mozilla Gecko 7
  • Arranxos en diferentes plataformas para mellor a seguranza e a estabilidade do produto

Máis doado iniciarse

  • Revisouse o asistente de creación de contas para mellorar a configuración do correo. Para abrilo, vaia ao menú Ficheiro, seleccione Novo, Conta de correo...
  • Nova páxina de resolución de erros para axudarlle cando atope un problema ao enviar e recibir correo. Pode atopalo no menú Axuda, Información para solucionar problemas.

Novo xestor de complementos

  • Vaia ao menú Ferramentas e seleccione Complementos para examinar todos os complementos dispoñíbeis para o Thunderbird.
  • Engada o Lightning Calendar
  • Engada a Vista de conversación, unha forma diferente de ver o correo

Máis novas funcionalidades para explorar

  • Thunderbird is based on the new Mozilla Gecko 7 engine
  • Several user interface fixes and improvements
  • Several fixes to attachment handling
  • Ability to print a summary of selected email messages
  • Platform improvements to Address Book
  • Fixed several security issues
  • Numerous platform fixes that improve speed, performance and stability
  • Vexa as notas da versión para obter máis información

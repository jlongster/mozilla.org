Děkujeme, že používáte Thunderbird!

Co je v Thunderbirdu nového?

Doufáme, že se vám budou nové funkce Thunderbirdu líbit. Svůj Thunderbird si můžete přizpůsobit s pomocí stovek doplňků do podoby, která vám vyhovuje.

Rychlejší, bezpečnější a více stabilní

  • Thunderbird má nyní rychlejší odezvu a startuje rychleji.
  • Thunderbird je založen na novém engine Mozilla Gecko 7.
  • Řada úprav platformy, které vylepšují bezpečnost a stabilitu produktu.

Snadnější začátek

  • Přepracovaný průvodce založením nového účtu je nyní chytřejší. Naleznete jej v nabídce Soubour pod Nový objekt -> Poštovní účet...
  • Nová stránka Technické informace vám pomůže s řešením problémů souvisejících s odesíláním a přijímáním e-mailů. Naleznete ji pod nabídkou Nápověda, kde je položka Technické informace.

Nový Správce doplňků

  • Pod nabídkou Nástroje zvolte položku Správce doplňků a začněte s prozkoumáváním doplňků pro Thunderbird.
  • Přidejte si kalendář Lightning.
  • Vyzkoušejte si Conversation View, který vám umožní zobrazovat vaše zprávy odlišným způsobem.

Další nové funkce, které můžete prozkoumat

  • Thunderbird is based on the new Mozilla Gecko 7 engine
  • Several user interface fixes and improvements
  • Several fixes to attachment handling
  • Ability to print a summary of selected email messages
  • Platform improvements to Address Book
  • Fixed several security issues
  • Numerous platform fixes that improve speed, performance and stability
  • Pro více informací se podívejte na poznámky k vydání.

תודה שבחרת להשתמש ב־Thunderbird!

מה חדש ב־Thunderbird?

אנו מקווים שתהנו מהתכונות החדשות ב־Thunderbird. באפשרותכם לבצע התאמה אישית לתוכנה כראות עיניכם, ובאמצעות מאות תוספות לבחירתכם, Thunderbird הוא כולו שלכם.

מהיר יותר, בטוח יותר, ויותר יציב

  • Thunderbird כעת יותר מגיב ומהיר יותר בעליה ובשימוש השוטף
  • Thunderbird כעת מבוסס על מנוע Mozilla Gecko 7 החדש
  • מספר תיקונים בפלטפורמה כדי לשפר את האבטחה ואת יציבות המוצר

קל יותר להתחיל

  • אשף יצירת חשבונות משופר, די להקל על תהליך הגדרת חשבון הדואר. ניתן למצוא אותו תחת תפריט קובץ, בחירה ב״חדש״, חשבון דואר אלקטרוני...
  • דף מידע לפתרון בעיות חדש כדי לסייע לכם כאשר תתקלו בבעיה במשלוח או בקבלה של דואר אלקטרוני. ניתן למצוא אותו תחת תפריט עזרה, מידע לפתרון בעיות.

מנהל תוספות חדש

  • תחת תפריט כלים, בחרו ״תוספות״ כדי לסייר בין כל התוספות הזמינות ל־Thunderbird.
  • הוסיפו את לוח השנה Lightning
  • הוסיפו את BiDi Mail UI כדי לקבל שליטה טובה יותר על תצוגה ושליחה של דואר אלקטרוני בעברית
  • הוסיפו את תצוגת השיחות כדי לראות את הדואר שלכם בדרך שונה
  • הוסיפו מילונים כדי לאפשר את פעולת בודק האיות בעברית, אנגלית ובשפות נוספות לבחירתכם

תכונות חדשות נוספות

  • Thunderbird is based on the new Mozilla Gecko 7 engine
  • Several user interface fixes and improvements
  • Several fixes to attachment handling
  • Ability to print a summary of selected email messages
  • Platform improvements to Address Book
  • Fixed several security issues
  • Numerous platform fixes that improve speed, performance and stability

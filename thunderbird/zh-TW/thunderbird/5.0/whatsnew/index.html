感謝您使用 Thunderbird!

Thunderbird 裡面有什麼新鮮事?

希望您能好好享受 Thunderbird 的新功能。您可以用任何您想要的方式來自訂 Thunderbird，並透過數百種不同的附加元件來讓 Thunderbird 更貼近您的需求！

更簡單就能入門

  • 改進的新增帳戶精靈改善了電子郵件的設定過程。您可以在檔案選單當中選擇新增、郵件帳號找到
  • 全新的疑難排解資訊頁面可以在您收發信遇到問題時幫助您。您可以在說明選單當中的疑難排解資訊找到。

全新的附加元件管理員

  • 在工具選單當中選擇附加元件以開始探索所有 Thunderbird 可用的附加元件。
  • 安裝 Lightning 行事曆
  • 安裝 Conversation View 以使用不同的方式來檢視您的郵件

更多新功能供您探索

  • 現在可以將分頁標籤重新排序或拖曳到其他視窗
  • 附件檔案大小現在會與附件一起顯示
  • 現在可以使用 RSS 資訊來源載入外掛程式
  • 有許多關於 Windows Vista 與 Windows 7 的佈景主題修復
  • 超過 390 個平台修復，讓速度、效能，以及穩定度變得更好
  • 看看 發行資訊 以取得更多資訊

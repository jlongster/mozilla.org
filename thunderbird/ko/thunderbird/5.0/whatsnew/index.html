Thunderbird 신규 기능

Thunderbird에 새로 탑재된 기능을 소개합니다. 여러분이 원하는대로 맞춤 기능을 제공하기 위해 수 백여가지의 부가 기능도 계속 제공합니다.

좀 더 편리해진 설정

  • 계정 설정 마법사를 좀 더 보완하여 쉽게 메일 설정이 가능하도록 했습니다. 파일 메뉴에서 새로 열기에서 메일 계정을 선택하십시오.
  • 메일 수신 시 문제가 발생했을 때 도움을 받을 수 있는 문제 해결 정보를 제공합니다. 도움말의 문제 해결 정보를 참조하십시오.

새 부가 기능 관리자

  • 도구 메뉴의 부가 기능을 선택하면 Thunderbird에 대한 새로운 부가 기능을 바로 검색할 수 있습니다.
  • Lightning 캘린더 추가하기
  • 대화형 모드 좀 더 특색있는 기능입니다.

세부 기능

  • 다른 창에서도 탭의 순서를 바꾸거나 드래그할 수 있음
  • 첨부 파일 크기에 따라 첨부를 분류할 수 있음
  • 플러그인이 RSS 피드를 읽을 수 있음
  • 윈도 비스타 및 윈도 7을 위한 테마 변경
  • 390여개의 성능 향상, 안정성 향상 기능 변경
  • See 제품 소개서(영문) 참고

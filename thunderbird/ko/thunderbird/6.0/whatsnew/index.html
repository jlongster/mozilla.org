Thunderbird 새 기능

Thunderbird의 새 기능을 즐겨 보십시오. 수 백개의 부가 기능으로 여러분만의 맞춤형 메일 프로그램을 만들 수 있습니다.

더 빠르고, 안전하고, 안정적인...

  • Thunderbird의 구동 시간을 더 줄이고 빠른 응답속도 개선
  • 신규 Mozilla Gecko 6 엔진 기반 개발
  • 다양한 성능 개선과 보안향상으로 안정성 확보

쉽게 시작하기

  • 계정 마법사를 새로 만들어 쉽게 이메일 설정을 하도록 변경하였습니다. 파일 메뉴의 새 메일 계정 설정을 눌러보세요.
  • 메일 주고 받을 때 생기는 문제점은 새로운 문제점 해결 페이지에서 확인 할 수 있습니다. 도움말 메뉴의 문제 정보를 확인해보세요.

새 부가 기능 관리자

  • 도규 메뉴 하단의 부가 기능을 선택하면 Thunderbird용 부가 기능을 볼 수 있습니다.
  • Lightning Calendar를 추가해 보십시오.
  • Conversation View를 추가하면 대화 모드로 볼 수 있습니다.

다양한 신규 기능

  • Windows 7용 테마 추가
  • Windows 7 작업 목록 추가
  • Microsoft Outlook에서 메일 가져오기 기능 향상
  • 최신 리눅스 배포판에 기본 메일 프로그램으로 탑재
  • 다양한 UI 향상
  • 더 자세한 정보는 제품 소개서(영문)을 참고하세요.

Thunderbird 3 주요 기능

Thunderbird 3을 사용해 주셔서 대단히 감사합니다.

Thunderbird 3 기능 소개

오랜 기간 개발해 온 Thunderbird의 새로운 기능들을 소개해 드리겠습니다. 검색 도구, 이메일 탭, 메시지 보관 및 수 백개의 부가 기능을 새로운 Thunderbird 3에서 선보입니다.

고급 필터링 도구를 이용한 검색

검색창에 키워드를 입력하면 새로운 탭에 검색 결과를 제공해 주며, 다양한 필터링 기능을 이용해 더 빠르게 검색 가능합니다.

이메일 탭 기능

Firefox의 탭 브라우징을 좋아하실 분이라면 이메일 탭 역시 마음에 드실 것입니다. 메일 항목을 더블 클릭하면 새로운 탭에 메일을 볼 수 있어 더 빠르고 편리하게 메일을 읽을 수 있습니다. 우클릭을 통해 메일함이나 메시지를 열수도 있습니다.

메시지 보관 기능

보관 버튼을 누르거나 'A'키를 누르기만 하면 메일을 보관 합니다. 보관을 결정한 메일은 보관 메일함에 안전하게 이동합니다.

기타 추가 기능:

  • 스마트 폴더
  • 활동 관리자
  • 원클릭 주소록
  • 첨부파일 알리미
  • 다양한 수백 개의 부가 기능들

더 자세한 사항은 Thunderbird 홈페이지를 참고하세요.

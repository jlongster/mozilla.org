感谢选用 Thunderbird!

Thunderbird 有什么新功能？

我们希望你能喜欢 Thunderbird 的全部新特性。通过数以百计的附加组件，你可以按照自己的意愿定制 Thunderbird，他完全是为你量身打造的。

更快，更安全，且更稳定

  • Thunderbird 启动和运行速度更快
  • Thunderbird 基于新的 Mozilla Gecko 7 引擎
  • 平台进行大量改进以提升安全和产品稳定性

易于上手

  • 改进的帐户创建向导，使邮件帐户设置更方便。他位于“文件”菜单下，选择“新建”，“邮件帐户...”
  • 在你收发邮件遭遇故障时，新的故障排除信息页面会帮助你。可以在“帮助”菜单，“故障排除信息”中找到他。

新的附加组件管理器

  • 在“工具”菜单中，选择“附加组件”项可以浏览所有 Thunderbird 可用的附加组件。
  • 添加 Lightning Calendar（日程任务管理）
  • 添加 Conversation View 以全然不同的会话方式查阅邮件

浏览更多新特性

  • Thunderbird is based on the new Mozilla Gecko 7 engine
  • Several user interface fixes and improvements
  • Several fixes to attachment handling
  • Ability to print a summary of selected email messages
  • Platform improvements to Address Book
  • Fixed several security issues
  • Numerous platform fixes that improve speed, performance and stability

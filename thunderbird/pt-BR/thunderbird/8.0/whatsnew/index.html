O que há de novo no Thunderbird?

Esperamos que você goste de todos os novos recursos do Thunderbird. Você pode personalizar o Thunderbird como quiser, e com centenas de complementos, o Thunderbird é todo seu.

Mais Rápido, Seguro e Estável

  • O Thunderbird está mais responsivo e mais rápido para abrir e usar
  • O Thunderbird é baseado no novo motor Mozilla Gecko 8
  • Inúmeras correções para melhorar sua segurança e estabilidade

Mais fácil de começar

  • Assistente de criação de contas revisado, para melhorar a configuração de emails. Encontre-o no menu Arquivo, selecione Nova conta de email...
  • Nova página 'Dados para suporte' para ajudá-lo quando você tiver problemas ao enviar ou receber mensagens. Encontre-a no menu Ajuda, Dados para suporte.

Novo gerenciador de complementos

  • No menu Ferramentas, selecione Complementos para começar a explorar todos os complementos disponíveis para o Thunderbird.
  • Adicionar o Calendário Lightning
  • Adicionar Conversation View, uma forma diferente de visualizar suas mensagens

Mais recursos novos para explorar

  • Novos atalhos para Pesquisar e Localizar
  • Melhor acessibilidade à lista de anexos
  • Diversas correções na manipulação de anexos
  • Complemento Folder Pane View Switcher
  • Leia o artigo Novidades do Thunderbird 8 para mais informações

¿Qué hay de nuevo Thunderbird?

Esperamos que disfrutes de todas las nuevas características de Thunderbird. Puedes personalizar Thunderbird como tú quieras, y con cientos de complementos para probar, Thunderbird es todo tuyo.

Más fácil de poner en marcha

  • Asistente de creación de cuentas revisado, para mejorar la configuración de cuentas. Encuéntralo seleccionando el menú Archivo, Nuevo, Cuenta de correo...
  • Nueva página de información sobre resolución de problemas para ayudarte cuando tengas problemas al enviar o recibir mensajes. Encuéntralo seleccionando el menú Ayuda, Información sobre resolución de problemas.

Nuevo administrador de complementos

  • En el menú Herramientas, selecciona Complementos para comenzar a explorar todos los complementos disponibles para Thunderbird.
  • Añade el Calendario Lightning
  • Añade Conversation View una manera diferente de ver tu correo

Otras características nuevas para explorar

  • Ahora las pestañas se pueden reordenar y arrastrar a ventanas diferentes
  • Ahora se muestran los tamaños de los adjuntos junto a ellos
  • Ahora se pueden cargar los plugins en los canales RSS
  • Hay varias correcciones en el tema gráfico para Windows Vista y Windows 7
  • Más de 390 correcciones en la plataforma que mejoran la velocidad, el rendimiento y la estabilidad
  • Visita las notas de versión para más detalles

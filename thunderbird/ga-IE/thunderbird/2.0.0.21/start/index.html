Tá an feidhmchlár ríomhphoist Thunderbird de chuid Mozilla níos cumhachtaí ná riamh. Tá sé níos fusa anois do chuid ríomhphoist a chur in eagar, a shlánú agus a shaincheapadh.

Mothaigh an difríocht. Tá Mozilla, comhphobal domhanda atá ag obair le chéile chun an tIdirlíon a fheabhsú do chách, ag forbairt agus ag tacú le Thunderbird.

Le haghaidh ceisteanna coitianta, nodanna agus cabhrach ginearálta, téigh go Ionad Cabhrach Thunderbird.

Le haghaidh eolais táirge, téigh go Leathanach Baile Thunderbird.

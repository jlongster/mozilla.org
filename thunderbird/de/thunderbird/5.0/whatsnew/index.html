Danke, dass Sie Thunderbird verwenden!

Was ist neu in Thunderbird?

Wir hoffen, dass Ihnen die ganzen neuen Funktionen in Thunderbird gefallen. Sie können Thunderbird aber anpassen, wie Sie möchten, und mit hunderten von Add-ons, die Sie ausprobieren können, gehört Thunderbird ganz Ihnen.

Einfacherer Start

  • Überarbeiteter Assistent zur Kontenerstellung, um das Einrichten von E-Mail-Konten zu verbessern. Sie finden ihn im Menü Datei, wählen Sie Neu > E-Mail-Konto…
  • Neue Seite mit Informationen zur Fehlerbehebung, um Ihnen zu helfen, wenn Sie beim Senden oder Empfangen von E-Mails Probleme haben. Sie finden sie im Menü Hilfe unter Informationen zur Fehlerbehebung.

Neuer Add-ons-Manager

  • Wählen Sie im Menü Extras den Punkt Add-ons, um zu beginnen, alle für Thunderbird verfügbaren Add-ons zu erkunden.
  • Fügen Sie den Lightning-Kalender hinzu.
  • Fügen Sie Conversation View hinzu, eine andere Möglichkeit, um Ihre E-Mails zu lesen.

Weitere neue Funktionen zum Entdecken

  • Tabs können jetzt neu angeordnet und in andere Fenster gezogen werden.
  • Die Größe eines Anhangs wird ab sofort zusätzlich zum Anhang angezeigt.
  • Plugins können jetzt in RSS-Feeds geladen werden.
  • Es gibt einige Fehlerkorrekturen für das Theme in Windows Vista und Windows 7.
  • Über 390 Fehlerkorrekturen, die Geschwindigkeit, Leistung und Stabilität verbessern.
  • Weitere Informationen finden Sie in den Versionshinweisen.

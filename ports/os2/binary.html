warpzilla binaries

Some binaries have been produced of the two Warpzilla versions. Note that Warpzilla has not been released yet, so every build you download is only work in progress. We do not guarantee anything so you are trying it at your own risk. This software may lead to data loss on your computer! For each version, you might find a history file that would show you what improvements or changes have been made.

complete viewers

  • M3 build
    (Milestone 3 build, including alpha level Java support)
  • Most recent viewer
    (GeckOS/2 0.003, released 1998-12-25, the "Christmas build" ;-)
  • a slightly older build of the nglayout browser had made it to Hobbes (seems to no longer be there though) and received another review in OS/2 e-Zine!
  • ngviewer (older version but still based on the nglayout engine)
  • Warpzilla 0.007 based on Mozilla Classic
    (is kind of old by now but interesting...)
For both, you will need to obtain and install the EMX run-time libraries if you haven't already done so for other software.

Testers can help us diagnose (and fix) bugs but please concentrate on the most recent version: our Classic version is now all but abandoned and we hope to be able to continue updating the new binary of the nglayout-based viewer soon.

components

A by-product of this porting work are small components of mostly mozilla source:

  • the nspr runtime
    (will eventually be required to shorten the download of the main viewer)
  • standalone javascript engine
  • independent JPEG Group library v.6b

These are emx-gcc appropriate builds. Check the directories for other compatibility formats.

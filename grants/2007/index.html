In 2007, Mozilla's grants nearly tripled, with over half a million dollars going toward projects that improve Web accessibility.

Summary of 2007 Grants

  • Accessibility: $533,000
  • Community: $242,250
  • Education: $100,000
  • Total: $875,250

Accessibility

  • $80,000 to NV Access to support an additional developer for the open source NVDA screen reader for Windows
  • $70,000 to the Adaptive Technology Resource Centre at the University of Toronto to implement accessibility features in the Dojo AJAX toolkit (phase 1)
  • $70,000 to the Adaptive Technology Resource Centre at the University of Toronto to implement accessibility features in the Dojo AJAX toolkit (phase 2)
  • $60,000 to Scott Haeger to implement Orca rich document browsing
  • $40,000 to Alexander Surkov to implement the IAccessible2 accessibility API for Windows in Mozilla
  • $35,000 to Eitan Isaacson to enhance the Orca open source screen reader for Linux to improve the quality of Braille output to match those of proprietary screen readers
  • $20,000 to Steve Lee for phase 4 of the Jambu project to optimize Firefox for pointer and switch access
  • $12,000 to Codethink Ltd to do a feasibility_study for migrating the AT-SPI accessibility interface to use D-Bus
  • $10,000 to Steve Lee for phase 2 of the Jambu project to optimize Firefox for pointer and switch access
  • $10,000 to Steve Lee for phase 3 of the Jambu project to optimize Firefox for pointer and switch access
  • $10,000 to Gijs Kruitbosch to enhance the accessibility of the ChatZilla IRC chat client used by Mozilla developers and others
  • $10,000 to the University of North Carolina Chapel Hill to support research into using commodity webcams as input devices for users with disabilities
  • $10,000 to Eitan Isaacson to develop the Accerciser test tool
  • $10,000 to Eitan Isaacson to improve the performance of Orca and implement other enhancements to it
  • $10,000 to NV Access to support development of the NVDA open source screenreader for Windows
  • $10,000 to Eitan Isaacson for the Orca AT-SPI library migration project
  • $10,000 to the GNOME Foundation to support GNOME accessibility projects and related activities
  • $9,500 to Ariel Rios to implement the AT-SPI Collection interface for better Firefox accessibility on Linux
  • $8,500 to Charles Chen to implement ARIA widgets in the Fire Vox open source screen reader extension for Firefox
  • $5,000 to Knowbility for sponsorship of 2008 accessibility-related events
  • $4,500 to Jörg Dotzki to develop a Firefox extension to simulate the experience of a color blind user in order to test the accessibility of web sites
  • $4,500 to Leo Spalteholz to enhance Firefox to develop enhanced keyboard navigation for users with disabilities and others
  • $1,000 to Simon Harper for sponsorship of the 2008 W4A conference
  • $1,000 to Knowbility for sponsorship of a Project:Possibility accessibility Code-A-Thon.
  • $1,000 to Web4All
  • $1,000 to the American Association of People with Disabilities for sponsorship of an AAPD event
  • $10,000 for CSUN 2007 booth and marketing expenses
  • $10,000 for travel scholarships for volunteers staffing the Mozilla booth at the CSUN 2007 conference

Community Grants

  • $100,000 to to the Participatory Culture Foundation
  • $100,000 to Creative Commons for general support of Creative Commons activities
  • $30,000 to the Mozdev Community Organization to support the mozdev.org site providing hosting of Mozilla-related open source projects
  • $30,000 to the Open Source Software Institute to enhance the OpenSSL cryptographic library and Apache mod_ssl SSL/TLS module
  • $15,000 to Henri Sivonen to support phase 1 of a project to develop an HTML5 parser in Java
  • $10,000 to Henri Sivonen to support phase 2 of a project to develop an HTML5 parser in Java
  • $10,000 to the Free Software Foundation to support development of the LinuxBIOS software and related activities
  • $10,000 to the Perl Foundation to support development of Perl 6 and related activities
  • $5,000 to Annodex Association to sponsor the Foundations of Open Media Software workshop in January 2008 in Melbourne, Australia
  • $2,250 to Peter Jaros to improve scriptability in the Camino browser and provide a new way to extend Camino using AppleScript
  • $30,000 for travel scholarships for the Internet as a Public Good symposium

Education Grants

  • $100,000 to Seneca College to continue support of Mozilla Education activities

The Mozilla Foundation is a member of the World Wide Web Consortium, and various Mozilla people represent Mozilla in W3C working groups and other W3C contexts. This grant was issued beyond Mozilla's existing W3C membership dues, and funded work on W3C CSS Validator work by giving to ERCIM, the W3C's donation program.

Grant Summary

  • 2009: $15,000
  • Focus: Open Source
  • Recipient: World Wide Web Consortium
  • Location: International

More Information

This grant allowed Mozilla to demonstrate its strong support for Web standards and open tools for web developers. By funding this project, we were able to contribute toward an effort that encourages Web developers to use good standards and have a reliable tool for improving their work.

W3C

The World Wide Web Consortium is an international community where Member organizations, a full-time staff, and the public work together to develop Web standards. Led by Web inventor Tim Berners-Lee, W3C's mission is to lead the Web to its full potential.

Please consider making a contribution to the Mozilla Foundation so we can continue to make grants to other worthwhile organizations such as the W3C.

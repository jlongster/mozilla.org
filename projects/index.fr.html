Nos projets

La communauté Mozilla crée beaucoup de logiciels et agit comme incubateur pour des idées innovantes dans le cadre de sa mission de bâtir un Internet meilleur.

Applications Mozilla

Ces applications sont développées par la communauté Mozilla et leur code est hébergé sur mozilla.org.

  • Firefox pour ordinateur

    Firefox®, le navigateur Web aux multiples récompenses, est sûr, rapide et a des fonctionnalités qui changeront votre manière d'utiliser le Web. N'acceptez rien de moins.

  • Firefox pour mobile

    En ajoutant Firefox® à votre téléphone portable, vous pouvez accéder à du contenu Web riche et apprécier vos fonctionnalités favorites de Firefox où que vous alliez.

  • Thunderbird

    Le client de messagerie sûr, rapide et facile, à la Mozilla. Le client de messagerie Thunderbird® comprend des filtres anti-spam intelligents, une recherche puissante et des vues personnalisables.

  • SeaMonkey

    SeaMonkey® est la suite tout-en-un, connue auparavant sous le nom de Suite Mozilla, contenant un navigateur Web, un client de messagerie et de groupes de discussion, un éditeur HTML, des outils de développement Web et un client IRC.

  • Lightning et Sunbird

    Lightning est une extension populaire permettant la gestion d'un agenda et de tâches. Sunbird® est une application multi-plateforme qui apporte la facilité d'utilisation Mozilla dans votre agenda.

  • Camino

    Camino est le navigateur Web optimisé pour Mac OS X avec une interface utilisateur en Cocoa et le puissant moteur de rendu Gecko. C'est le navigateur simple, sûr et rapide pour Mac OS X.

Projets Mozilla Drumbeat

Drumbeat est une communauté de Mozilliens qui utilisent la technologie Web dans de nouvelles voies pour comprendre la vie numérique, y participer et en prendre le contrôle.

  • Universal Subtitles

    Universal Subtitles est un projet de logiciel et une communauté qui crée l'outil de sous-titrage le plus simple, libre et ouvert à tous, pour ajouter des sous-titres sur une vidéo.

  • WebMadeMovies

    WebMadeMovies est un laboratoire pour la vidéo ouverte et un studio de production qui rassemble les réalisateurs et les développeurs Web les plus innovants pour montrer ce dont la vidéo ouverte est capable.

Voir d'autres projets Drumbeat…

Les laboratoires Mozilla

Mozilla Labs est un laboratoire virtuel où les gens viennent pour créer, expérimenter et jouer avec de nouvelles innovations et technologies Web.

  • Raindrop

    La mission de Raindrop : rendre attrayante la participation à des conversations avec les gens que vous voulez, que ces conversations soient par courriel, sur twitter, le blog d'un ami ou sur un site de réseau social.

  • Skywriter

    Mozilla Skywriter (anciennement appelé Bespin) est un éditeur de code rapide, personnalisable, écrit pour les navigateurs Web modernes, qui vous permet de coder dans le nuage.

Voir d'autres expérimentations des Mozilla Labs…

Outils pour développeur de Mozilla

La communauté Mozilla crée beaucoup d'outils utilisés pour le développement de logiciels.

  • Bugzilla

    Bugzilla™ est un système de suivi de bogues conçu pour aider les équipes à gérer le développement des logiciels. Des centaines d'organisations dans le monde utilisent cet outil très puissant pour s'organiser et communiquer efficacement.

  • Firebug

    Firebug est l'outil le plus populaire et le plus puissant pour le développement Web qui vous permet d'examiner, déboguer et analyser vos développements dans votre navigateur.

Applications basées sur Mozilla

Ces applications sont développées par des particuliers ou des organisations en utilisant les technologies Mozilla. Si vous voulez proposer d'autres applications à indiquer ici, veuillez nous le faire savoir.

  • Instantbird

    Instantbird est un client de messagerie instantanée multi-protocole qui utilise le moteur de rendu de Mozilla et la bibliothèque libpurple de Pidgin. Vous pouvez avec lui vous connecter à tous vos comptes de messagerie instantanée.

  • Snapstick

    Snapstick combine la facilité de découverte du contenu de votre ordinateur portable, de votre tablette ou de votre téléphone portable avec le confort de visualisation d'un grand écran de télévision.

Voir d'autres applications basées sur Mozilla…

Technologies Mozilla

La communauté Mozilla crée différentes technologies disponibles gratuitement pour les particuliers ou les organisations qui peuvent les utiliser dans leurs propres produits.

  • Gecko

    Gecko est le moteur de rendu qui lit les contenus Web, tels que HTML, CSS, XUL et JavaScript et les restitue sur l'écran de l'utilisateur. Dans les applications basées sur XUL, Gecko est utilisé pour afficher l'interface utilisateur également.

  • XULRunner

    XULRunner fournit un environnement aux développeurs pour construire des applications basées sur XUL, telles que Firefox et Thunderbird. Il propose des mécanismes pour l'installation,la mise à jour et la désinstallation d'applications.

Voir d'autres technologies Mozilla…

Projets archivés

Mozilla crée des logiciels depuis 1998, et certains de ces projets ne sont plus actifs.

  • Minimo

    Minimo fut la première tentative de création d'un navigateur pour équipements mobiles. Le développement a été stoppé en 2007 et peu après, le projet Firefox pour mobile a démarré.

  • Suite Mozilla

    La Suite Mozilla comprenait un ensemble d'applications qui étaient publiées au commencement du projet Mozilla. En 2006, le développement a été arrêté quand la Fondation Mozilla s'est concentrée sur les projets Firefox et Thunderbird.

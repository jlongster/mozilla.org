Lightning System Requirements

Lightning 1.0 beta4 is compatible with Thunderbird 5.0 and SeaMonkey 2.2.

Windows

Operating Systems
  • Windows XP
  • Windows Server 2003
  • Windows Vista
  • Windows 7
Minimum Hardware
  • Pentium 233 MHz (Recommended: Pentium III 500MHz or greater)
  • Windows 7, Windows Vista, Windows XP: 786 MB RAM (Recommended: 1GB RAM or greater)
  • 10 MB hard drive space

Mac OS X

Operating Systems

Mac OS X 10.4.x and later

Minimum Hardware
  • Macintosh computer with an Intel x86 processor
  • 256 MB RAM (Recommended: 512 MB RAM or greater)
  • 10 MB hard drive space

Linux

Software Requirements

Please note that Linux distributors may provide packages for your distribution which have different requirements.

  • Linux kernel - 2.2.14 or higher with the following libraries or packages:
    • glibc 2.3.2 or higher
    • XFree86-3.3.6 or higher
    • gtk+2.0 or higher
    • fontconfig (also known as xft)
    • libstdc++5
      (Many modern Linux distributions only package libstdc++6, which is incompatible with Lightning. Therefore please install the package "libstdc++5" or "compat-libstdc++" on your system before installing Lightning)
Minimum Hardware
  • Intel Pentium II or AMD K6-III+ 233 MHz CPU (Recommended: 500MHz or greater)
  • 64 MB RAM (Recommended: 128 MB RAM or greater)
  • 10 MB hard drive space

Sunbird 1.0 beta1

Mozilla Sunbird® is a cross-platform calendar application, built upon Mozilla Toolkit. Our goal is to provide you with a full-featured and easy to use calendar application that you can use around the world.

This is the last public Sunbird release by the Calendar Project.

If you would like to send us your calendar files, please follow these steps:

  1. Create a new local calendar, e.g. "$Country$ Holidays"
  2. Create the different holidays of your country in this local calendar
  3. Right-click on this calendar and choose "Export Calendar". Please use the iCalendar (ics) format.
  4. Open a bug in Bugzilla in the Website component
  5. Add your holiday file as an attachment in the new bug

usage: mozDriver.pl [<options>]
(-b|--bugurl)Bugzilla URL.
(default is http://bugzilla.mozilla.org/show_bug.cgi?id=)
(-e|--engine) <type> ... Specify the type of engine(s) to test. <type> is one or more of (seamonkey).
(-f|--file) <file> Redirect output to file named <file>.
(default is results-<engine-type>-<date-stamp>.html)
(-h|--help)Print this message.
(-k|--confail)Log failures to console (also.)
(-l|--list) <file> ...List of tests to execute.
(-L|--neglist) <file> ... List of tests to skip.
(-o|--opt) <options> Options to pass to the JavaScript engine.
(Make sure to quote them!)
(-p|--testpath) <path>Root of the test suite. (default is ./)
(-s|--shellpath) <path>Location of JavaScript shell.
(-t|--trace)Trace script execution.
(-u|--lxrurl) <url>Complete URL to tests subdirectory on lxr.
(default is http://lxr.mozilla.org/mozilla/source/js/tests/)
(-x|--noexitmunge)Don't do exit code munging (try this if it
seems like your exit codes are turning up
as exit signals.)

Is NSS FIPS Compliant?
NSS contains a FIPS-certified software cryptographic module. See the NSS FIPS page for the FIPS validation history of the NSS software cryptographic module.
Is there any sample code and documentation?
NSS documentation is listed in the Documentation section on the main NSS page. Among them, The SSL Reference, NSS Tools manuals, and NSS Technical Notes are especially useful.

NSS example code is essentially developer test code; with that understanding, the best directory to look for sample code is in the mozilla/security/nss/cmd directory:

http://lxr.mozilla.org/mozilla/source/security/nss/cmd

SSL examples:

selfserv
strsclnt
tstclnt

How can I debug my SSL connection?
By using the NSS tool ssltap.
Where can I find a listing of NSS error codes?
Error codes that may be returned by NSS functions are listed in three header files. The error codes fall in three numeric ranges:
  -12288 .. -11265    (-12k .. -12k+1k-1)    SSL errors
   -8192 ..  -7169    (-8k .. -8k+1k-1)    NSS ("SEC") errors
   -6000 .. -5001                                     NSPR errors
These three URLs show error strings for those error numbers.

http://www.mozilla.org/projects/security/pki/nss/ref/ssl/sslerr.html#1040263
http://www.mozilla.org/projects/security/pki/nss/ref/ssl/sslerr.html#1039257
http://bonsai.mozilla.org/cvsblame.cgi?file=mozilla/nsprpub/pr/include/prerr.h
Can I import OpenSSL's PEM files into NSS?
PEM is a file format defined by OpenSSL, based on ideas found in the PEM RFCs. Basically it's a text file containing one or more items such as certificates, keys, etc., each of which is base64 enocded, and surrounded by text lines that look like 
------- BEGIN CERTIFICATE ------
(base64 lines here)
------- END CERTIFICATE ------
For things other than certificates, the word CERTIFICATE is replaced in those text files with a word that describes those other things. NSS supports files containing only a single PEM-format certificate. certutil will import them and create them when the -a argument is used.

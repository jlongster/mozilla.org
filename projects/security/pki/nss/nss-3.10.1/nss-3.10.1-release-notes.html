Introduction

Network Security Services (NSS) 3.10.1 is a patch release for NSS 3.11. The bug fixes in NSS 3.10.1 are described in the "Bugs Fixed" section below.

Distribution Information

The CVS tag for the NSS 3.10.1 release is NSS_3_10_1_RTM.  NSS 3.10.1 requires NSPR 4.5.2.

See the Documentation section for the build instructions.

NSS 3.10.1 source and binary distributions are also available on ftp.mozilla.org for secure HTTPS download:

  • Source tarballs: https://ftp.mozilla.org/pub/mozilla.org/security/nss/releases/NSS_3_10_1_RTM/src/.
  • Binary distributions: https://ftp.mozilla.org/pub/mozilla.org/security/nss/releases/NSS_3_10_1_RTM/. Both debug and optimized builds are provided. Go to the subdirectory for your platform, DBG (debug) or OPT (optimized), to get the tar.gz or zip file. The tar.gz or zip file expands to an nss-3.10.1 directory containing three subdirectories:
    • include - NSS header files
    • lib - NSS shared libraries
    • bin - NSS Tools and test programs
You also need to download the NSPR 4.5.2 binary distributions to get the NSPR 4.5.2 header files and shared libraries, which NSS 3.10.1 requires. NSPR 4.5.2 binary distributions are in https://ftp.mozilla.org/pub/mozilla.org/nspr/releases/v4.5.2/.

Bugs Fixed

The following bugs have been fixed in NSS 3.10.1.
  • Bug 301212: Upgrade mozilla/security/nss/cmd/zlib to zlib 1.2.3
  • Bug 302560: NSS 3.10.1 needs only zlib fix

Documentation

For a list of the primary NSS documentation pages on mozilla.org, see NSS Documentation.

Compatibility

NSS 3.10.1 shared libraries are backward compatible with all older NSS 3.x shared libraries. A program linked with older NSS 3.x shared libraries will work with NSS 3.10.1 shared libraries without recompiling or relinking.  Furthermore, applications that restrict their use of NSS APIs to the functions listed in NSS Public Functions will remain compatible with future versions of the NSS shared libraries.

Feedback

Bugs discovered should be reported by filing a bug report with mozilla.org Bugzilla (product NSS).

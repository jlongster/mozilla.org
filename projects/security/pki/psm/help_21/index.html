The files linked from this page must be viewed in Netscape 6/Mozilla 0.9 or later versions.

Status

Cleanup and corrections are underway for the Privacy & Security help files that shipped with Mozilla 0.9.2/Netscape 6.1.

Changes checked in to date include:

  • What to Do If You Forget Your Password. Rewritten for new Reset Master Password button.
  • Managing Certificates That Identify You. Changes reflect the fact that you can't back up certificates and keys stored on smart cards/external security devices.
  • Managing CRLs. Expanded to include information about the Next Update field.
  • Managing Profiles. New section.
  • Privacy & Security Preferences - Cookies. New section.
  • Privacy & Security Preferences - Forms. New section.
  • Privacy & Security Preferences - Master Passwords - Change Master Password. Changes reflect changed UI.
  • Privacy & Security Preferences - Master Passwords - Reset Master Password. New section for new UI.
  • Privacy & Security Preferences - SSL. Revised TLS info (proxy bug is fixed) and SSL Warnings (names of options have changed, one has been added).
  • Your Certificates. Changes reflect the fact that you can't back up certificates and keys stored on smart cards/external security devices. Changes to Delete My Certificate section reflect changes in UI and help button targeting.
  • Edit Web Site Certificate Trust Settings. Changes reflect changes to the UI.
  • Edit CA Certificate Trust Settings. Changes reflect changes to the UI.
  • New Web Site Certificate. Changes reflect changes to the UI.
  • Browser Won't Accept CRL. New section for alert encountered when the client's available CRL for a server cert has a next update due before the current time.
  • Glossary. Changes to definitions of security device, software security device.
  • Changes throughout to "double-click to expand the list" rather than "click to expand the list."
  • Various minor formatting and text corrections, including some changes to headings and the TOC for consistency.

Summary Action
Start Start the Mozilla Firefox browser. The browser should start.
Installation
  1. On Windows and Linux, Launch the Install wizard.
  2. Navigate through the wizard with the Next buttons.
  3. Accept the licensing agreement and perform a "Standard" installation.
  4. When the installation has completed, ensure that the checkbox to launch the application is checked and click the "Finish" button.
  5. On Mac, drag the Thunderbird package to your Application folder.
 The installer should be complete, all of it's content should fit properly in the wizard panels, and the application should launch at completion of the installation.

Group: 02. Firefox Browser-Functionality

Summary Action
Address Field In the address field, type and visit a URL of choice. (If you can't decide, try http://texturizer.net/firefox/.) The specified page should load.
Open File Use File | Open to open a local html, text, or image file. File should be displayed in browser window.
Save page
  1. Visit http://www.mozilla.org/projects/firefox
  2. From the toplevel menu, select File | Save Page As
  3. In the resulting file picker, select "Web Page, complete".
  4. Choose a target directory (e.g., your desktop or home directory).
  5. Click the Save button.
  1. Verify that the file Firefox Browser Project Page.html is saved, as well as folder called 'Firefox Browser Project Page_files' which contains 3 files: browser.css, mozilla-banner.gif and persistant-style.css
  2. Open the local file Firefox Browser Project Page.html and verify that it looks the same as http://www.mozilla.org/projects/firefox
New Window Open a new browser window using Ctrl + N or File | New Window. New Browser Window should open.
Top Site Open http://www.yahoo.com. Ensure that there are no obvious layout problems. Page should display as expected.
Top Site Load http://www.msn.com. Check for layout problems. Page should display as expected.
Top Site Visit http://www.amazon.com. Check for layout problems Page should display as expected.
Top Site, Popup Blocking/unblocking
  1. Visit http://www.cnn.com.
  2. A message below tab bar should appear to inform you that Firefox has blocked a popup.
  3. An [X] icon should appear in the status toolbar.
  4. Use Tools | Options | web features to add www.cnn.com to "allowed sites" list. Or select the [X] icon | Allow popups for www.cnn.com. Reload the page. Check to see that the popup appears.
  1. Initially no popups should appear
  2. After step 4, the page should display as expected and the popup should also appear.
FTP Load ftp.mozilla.org; navigate to ftp://ftp.mozilla.org/pub/mozilla.org/firefox/nightly/ Specified directory should display.
HTTPS
  1. Visit the secure site https://sourceforge.net.
  2. Observe that a Warning Dialog comes up,press OK to continue.
  3. Verify that the Lock icon at the bottom right side of the browser window appears locked.
  4. Visit http://sourceforge.net. The Lock icon at the bottom right side of the browser window should appear unlocked.
  5. Visit https://sf.net and ensure that the "Security Error: Domain Name Mismatch" dialog appears.
  6. View the certificate, ensure that it is for sourceforge.net and looks "correct".
  7. Click Okay to visit the site, ensure that it loads the same https://sourceforge.net as above.
 Correct identification of secure site.
Page Load Controls
  1. Visit two successive sites.
  2. Click Back button twice.
  3. Click Forward button twice.
  4. Begin to load www.apple.com in the browser, Click the Stop button before the page load is completed. Page loading and any indication of loading should cease.
  5. Click Reload and ensure that the page loads completely
 Page loading should occur as expected.
Search Focus search field with mouse or Ctrl + E, type a search term and hit enter. Search results should be displayed.
Bookmarks
  1. Use Bookmarks | Add to Bookmarks... and hit OK in the resulting dialog to add a bookmark.
  1. Select Bookmarks | Manage Bookmarks

  3. alternatively, use Ctrl + B to open bookmarks in the sidebar. Find the new bookmark and double-click to open.
Options
  1. Select Tools | Options to bring up the Options window.
  2. Select the General tab and click the Use Current Page button in Home Page section.
  3. Click OK to save changes and dismiss the Options window.
 Click the Home button on the navigation toolbar and verify that it loads the new home page.

    Group: 03. Firefox Browser-Uninstall

    Summary Action
    Quit Use File | exit to quit the browser. Firefox should shut down.
    Shutdown and Uninstall
    1. Select the File | Exit menu item
    2. On Windows, go to Start -> Control Panel -> Add or Remove Programs, select Firefox and click Uninstall.
    3. On Mac, drag the Firefox application package to the trash.
    4. On Linux, delete the Firefox folder.
    		 The application should be removed but the profile should remain intact.
